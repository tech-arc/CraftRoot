## 📌 Project Overview
CraftRoots is a modern Direct-to-Consumer (D2C) e-commerce platform designed to bridge the gap between skilled Indian rural artisans/manufacturers and conscious global buyers. 

The traditional handicraft supply chain relies on layers of middlemen, inflating retail prices while leaving the original creators with less than 15% of the profit. CraftRoots solves this by providing a secure, direct marketplace, ensuring fair compensation for makers and authentic goods at factory prices for consumers.

### 🎯 Key Features
* **Zero Middlemen:** Buyers purchase directly from verified artisans.
* **Interactive Data Visualization:** Includes real-time charts (Chart.js) demonstrating the financial impact of the D2C model vs. traditional retail.
* **Modern UI/UX:** Built with a fully responsive, custom Glassmorphism design system using Tailwind CSS.
* **E-commerce Flow:** Features a complete shopping experience including product filtering, dynamic cart updates, and wishlist management.
* **Secure Delivery Focus:** Emphasizes safe, insured shipping directly from rural workshops.
